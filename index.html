The Most Common HTML Tags and Attributes

HTML Tags

Tag

Name

Description

Comments
<body> Body The key HTML element that represents the content of an HTML document. There can be only one <body> element in a document.
<h1> to <h6> Heading The elements that represent six levels of section headings. <h1> is the highest section level and <h6> is the lowest.
<p> Paragraph The element that represents a paragraph.
<span> Span A generic inline container for phrasing content. It can be used to group elements for styling purposes (using the class or id attributes), or because they share attribute values, such as lang. It should be used only when no other semantic element is appropriate.
<div> Division The generic container for flow content. <div> has no effect on the content or layout until styled in some way using CSS.
<em> Emphasis The element that marks text that has stress emphasis. The <em> element can be nested, with each level of nesting indicating a greater degree of emphasis.
<strong> Strong Importance The element that indicates that its contents have strong importance, seriousness, or urgency. Browsers typically render the contents in bold type.
<br> Line Break The element that produces a line break in text. It is useful for writing a poem or an address, where the division of lines is significant.
<ol> Ordered List The element that represents an ordered list of items. The list is typically rendered as a numbered list.
<ul> Unordered List The element that represents an unordered list of items. The list is typically rendered as a bulleted list.
<li> List Item An item in a list. It must be contained in a parent element: an ordered list, an unordered list, or a menu.
<image> Image Embed The element that embeds an image into the document. A self-closing tag.

class Class The global attribute, a space-separated list of the case-sensitive classes of the element. Classes allow CSS and JavaScript to select and access specific elements via the class selectors or functions.
